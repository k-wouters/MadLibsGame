<h1 id="mad-libs-style-game">Mad-Libs Style Game</h1>
<p><strong>Website Description:</strong> A Mad-Libs style game created for the AP CSA Create Task. The program displays six different stories, where clicking on a story displays a list of inputs. The user fills out answers and clicks submit to show the completed story. </p>
<p><strong>Deployed Web Page:</strong> <a href="https://www.terpconnect.umd.edu/~katywou/Mad-Libs-Game/">https://www.terpconnect.umd.edu/~katywou/Mad-Libs-Game/</a></p>
